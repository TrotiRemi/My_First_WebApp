Placez ici votre logo `logo.png`.

Recommandations rapides:
- Format: PNG carré, 256×256 px ou 512×512 px
- Fond: transparent de préférence
- Couleurs suggérées: beige (#f5ede3), bleu ciel (#76c6ea), orange (#f28c28)

Note: Si le fichier `IMG/logo.png` est manquant, le site affichera automatiquement un logo SVG de secours dans le header.
